# Worklog — AI in Data Analytics: Northwind Project

## 2026-07-08

### Отклонено (rejected)
**Предложение AI:** использовать `pd.read_csv(..., on_bad_lines="warn")`, чтобы обойти ошибку парсинга в `customers.csv`, `suppliers.csv` и `orders.csv`, просто пропуская проблемные строки.
**Почему отклонено:** это скрыло бы потерю данных без понимания причины ошибки — быстрое решение вместо разбора первопричины.

### Принято (accepted)
**Предложение AI:** написать функцию `read_csv_fix_address_commas`, которая восстанавливает адресные поля, "разорванные" незаэкранированной запятой, вместо того чтобы отбрасывать такие строки.
**Почему принято:** решение основано на подтверждённой причине ошибки (незаэкранированная запятая в адресе, а не проблема кодировки), не теряет ни одной строки данных; проверено полным запуском ноутбука без ошибок.
